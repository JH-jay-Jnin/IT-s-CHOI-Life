변수 선언 방식

        var   : 변수, 변수명 중복 O (중복시, 덮어쓰기), 함수 레벨 
        let   : 변수, 변수명 중복 X (중복시, 오류!!!!), 블록{} 레벨 scope
        const : 상수, 변수명 중복 X                   , 블록{} 레벨 scope

        

        1순위 : const
        ( JS는 특정 요소를 선택해서 사용하는 경우가 많아서 상수에 고정시켜둠 ) 
        
        2순위 : let
        
        3순위 : var ( 거의 안씀 )
